Set up

Copy dist config file:
$ cp core/phpunit.xml.dist core/phpunit.xml
Edit and add your environment variables: 
$ vim core/phpunit.xml
<env name="SIMPLETEST_BASE_URL" value="mysql://root@localhost/d8"/>
<env name="SIMPLETEST_DB" value="http://d8.dev"/>
Or run through command line: 
$ export SIMPLETEST_DB=mysql://root@localhost/d8
$ export SIMPLETEST_BASE_URL=http://d8.dev
Run: 
$ phpunit -c core/phpunit.xml --group=user --testsuite=unit
